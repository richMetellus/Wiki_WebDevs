Step 2: Build Website with WordPress 
=======================================

Start building your website:

#. On the hostinger dasbord, click on ``Hosting``

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.0_HostingerDashboard.png
        :width: 850px 

#. Click on ``Setup`` on the *Business Web Hosting* card.
#. Then this will ask you if you want to *Create* or *Migrate* a website, select

#. On the next page, select ``Wordpress with AI``

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.3_HostingerDashboard_Wordpress.png
        :width: 850px

        the first option, but if you want to build a website without all the 
        functionality of WordPress then you can also go with the **Hostinger Website Builder**.

        * The **Hostinger Website Builder** is a little less customizable since 
          you don't have **WordPress** but it's easier.

#. Click on ``Create with WordPress``
#. Next, you will need to create an WordPress admin account. Fill in the details 
   for this account.

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.4_HostingerDashboard_WordpressAdminAccount.png
        :width: 850px

#. Next, select the type of website you want to build then > Next

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.3_HostingerDashboard_WordpressAdminAccount.png
        :width: 850px

    * In my use case, the wesbsite type is ``Buisiness and services``

        .. image:: ./_images/JNL-003.0_KSTWebsite_1.0_HostingerKST_WebsiteType.png
            :width: 650px 

#. For your website you will have options such as **Pre-build templates** or **Standard themes**

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.4_HostingerDashboard_PrebuildTheme.png
        :width: 850px 
    
    * Selected *Pre-build themes* as it easier for the purpose of the tutorial.

    .. image:: ./_images/JNL-003.0_KSTWebsite_1.1_HostingerKST_Pre-buildWebsitesAndThemes.png

        :width: 850px

#. Next, chose the pre-selected plugins

    .. image:: ./_images/JNL-002.0_WordPressCharlie_2.6_HostingerDashboard_Pluggins.png

#. Select the content for your business.
