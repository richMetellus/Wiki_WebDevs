Web Development 
################

**Intro**

To get started with learning web technologies, I set to understand the tools used 
to create modern websites. There are so many web builders tool and choosing one 
can be daunting. 

* Will the technology or framework be relevant tomorrow?
* Are the skills transferrable?
* What is the ecosystem like?

**Context** Why I am looking into this:

I want to create a company agency while not having much capital. I paid others to 
design but they never quite deliver and almost got scammed. So I had to learn 
to build skills like video editing, creative media design, a landing page about 
the company, and create automation around the business operation. (newsletter,
email marketing, phone business setup). This is why I started documenting the 
process and research I found online.

I first started by prompting chatGPT on what is relevant today. The answer 
can be found in journal entry :ref:`Web technology tooling and framework AI prompt <web-tech-AI-Answer-target>`
but the answer was not quite right.
